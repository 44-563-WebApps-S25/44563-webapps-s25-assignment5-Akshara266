Which row breaks first?

When resizing the page, the first row (letters) breaks first. This is because the 'col-sm-2' class is applied to the first row, which makes it more prone to wrapping on smaller screens, whereas the second row (pet names) uses the 'col-md' class, which is designed to remain more flexible across medium-sized screens.
